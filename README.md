# Toolkit

A dark-themed developer utility toolkit — all in one page, no backend required.

## Tools

- **JSON Formatter** — validate, format, and minify JSON
- **Base64 Encode/Decode** — convert text to/from Base64
- **Regex Tester** — live regex matching with flags
- **URL Encode/Decode** — encode/decode URL components
- **Password Generator** — cryptographically secure random passwords
- **Color Picker** — convert between HEX, RGB, HSL, and CMYK
- **Hash Generator** — SHA-256, SHA-384, SHA-512 hashes
- **Text Diff** — line-by-line text comparison

## Setup

Zero dependencies. Just open `index.html` in a browser or deploy to GitHub Pages.

## Deploy to GitHub Pages

1. Enable GitHub Pages in repo settings: `Settings -> Pages -> Source: main branch`
2. Your site will be live at `https://anthonyjr.github.io/toolkit/`
